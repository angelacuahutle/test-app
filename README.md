- Rails 7..2
- Ruby 3.1.0
- mysql 8
- Rspec 
- Capybara
- Tailwind

- Hotwire/Stimulus/TurboFrame, on the front end, without having to go back to the server to refresh the screen, or using any JS libraries (such as datatables).
